# Project Setup Guide

## Installation

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd <project-folder>
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

## Compile and Run the Project

```bash
# development
$ npm run dev

# build the project
$ npm run build

# build the production build
$ npm run preview

```
